Der out-Ordner

Außerdem wurde von der IDE das out-Directory erstellt. Dieses enthält alle von IntelliJ kompilierten Klassen.

Fun-Fact Öffnet man eine der Klassen im out-Directory, findet man einen Kommentar: Source code recreated from a .class file by IntelliJ IDEA (powered by Fernflower decompiler).

Ein Decompiler wandelt Bytecode zurück in lesbaren Code. Man kann ihn also dafür nutzen, sich eine Bibliothek anzusehen, auf deren Quellcode man keinen Zugriff hat. Vor 2014 musste man sich dafür ein Plugin installieren.
